GAME DAY THREADS — GITHUB PAGES DEPLOY
========================================

This folder is ready to drop into GitHub Pages.
The home page is named index.html so GitHub serves it automatically.

YOUR URL WILL BE:
-----------------
https://YOUR-USERNAME.github.io/REPO-NAME/

For example, if your username is "carlos" and you name the repo
"gameday-threads", your live URL is:
https://carlos.github.io/gameday-threads/

STEP-BY-STEP DEPLOY:
--------------------
1. Go to github.com — sign in or create a free account
2. Click the green "New" button (or visit github.com/new)
3. Name your repo: gameday-threads
4. Set to PUBLIC (required for free GitHub Pages)
5. Check "Add a README file"
6. Click "Create repository"

7. On the new repo page, click "Add file" → "Upload files"
8. Drag all FOUR files from this folder into the upload box:
   - index.html
   - design.html
   - football.html
   - team-orders.html
9. Scroll down and click "Commit changes"

10. Click the "Settings" tab (top of the repo)
11. Click "Pages" in the left sidebar
12. Under "Build and deployment" → "Source":
    Select: "Deploy from a branch"
    Branch: main / (root)
    Click Save

13. Wait 1-2 minutes. Refresh the Pages settings page.
    You'll see a green box with your live URL.

14. Click it — your site is live!

UPDATING THE SITE LATER:
------------------------
- Re-upload the changed file via "Add file" → "Upload files"
- GitHub asks "Replace existing?" — say yes
- Live site updates in ~1 minute

THE FOUR PAGES:
---------------
• index.html        ← Home page (the one visitors land on)
• design.html       ← Design Your Own custom order form
• football.html     ← Football Season landing page
• team-orders.html  ← Team & Booster Orders page

All four pages are interlinked. Nav, footer, and CTAs work across them.
